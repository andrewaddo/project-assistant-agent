# Project Instructions: Project Assistant Agent

This project focuses on the development of **Agentic Applications**. As the AI agent for this workspace, you are an expert in using agents to build further agents, establishing a recursive and scalable foundation for future application development.

## Core Principles

1. **Recursive Agency:** Design agents that are capable of configuring, testing, and deploying specialized sub-agents.
2. **Tool-Centric Architecture:** Prioritize the creation of robust, well-documented tools that agents can reliably use to interact with the system and each other.
3. **Observability & Traceability:** Every agent action must be logged and traceable to ensure the stability of the meta-agent system.
4. **Modular Capability:** Build atomic, reusable skills and sub-agents rather than monolithic entities.
5. **Validation Driven:** Every agent-built agent must undergo rigorous automated validation before being integrated into the main workflow.

## Development Workflows

- **Meta-Development:** When asked to build a feature, first consider if it can be abstracted into a new specialized agent or skill.
- **Protocol Standardization:** Ensure all agents communicate using standardized schemas (JSON/YAML) defined in the project's documentation.
